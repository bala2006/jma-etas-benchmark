# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening an issue on the GitHub repository.

## Security Best Practices

- This is a scientific computing project using Python
- Follow standard data security practices when handling earthquake data
- Ensure proper attribution when using the ETAS model

## Contact

For security-related concerns, please open an issue on GitHub.
