# Contributing

Thank you for your interest in contributing to JMA ETAS Benchmark.

Contributions are welcome in the following areas:

- Temporal validation and backtesting
- Additional forecasting baselines
- Documentation improvements
- Performance optimization
- Visualization improvements
- Reproducibility and CI

## Development

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

Please include clear descriptions of scientific or methodological changes.
